Rotcustom

Automatically decode and encode rotation ciphers online.

Settings

Please input your character offset and your message you want to encrypt/decrypt in the boxes. If you don't know what to do, hover over the "What is this?" text next to the inputs. To decode ciphers that you have made, take your original offset (lets say, 4), and subtract it from 26. To decode a cipher from the offset 26, you would use offset 22.
[ What is this? ]Input how many letters you want to shift the text by. See settings description for how to use negative ciphers.


[ What is this? ]Input the encoded cipher, or the message you want to decode here. It doesn't matter which, as long as the offset is correct, because it will rotate the letters either way.
